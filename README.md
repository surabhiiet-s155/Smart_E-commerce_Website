# Smart_E-commerce_Website
“Smart Step Store – a beginner-friendly e-commerce website using HTML and CSS.”

**E-Commerce Website**

A simple and clean e-commerce website created using HTML and CSS.
This project displays products in a basic layout similar to an online store.

Features

- Simple product listing layout
- Clean and minimal design
- Product cards with name and price
- Responsive and beginner-friendly structure

Technologies Used

- HTML5
- CSS3

Project Structure

ecommerce-website
~index.html
~style.css
~README.md

How to Run

1. Download or clone the repository
2. Open "index.html" in your browser
